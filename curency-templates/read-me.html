This folder contains templates for printing Euro, British Pound and other special currency symbols.

++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

How to use currency templates:

There are three custom templates file and one system backup file(default).

1. Euro

To add euro symbol on printable matter, please copy files from euro folder and paste them on includes folder.

2. Pound

To add pound symbol on printable matter, please copy files from euro folder and paste them on includes folder.

3. Custom

To add other custom currency like Yen or other follow these steps:

(i) Grab unicode-decimal code of your currency
(ii) Paste code in express.php and express-rec.php, you will find some stars and a comment for where to paste

4. Default

To reset printable matter, please copy files from default folder and paste them on includes folder.
